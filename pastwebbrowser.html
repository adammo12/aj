Page History Explorer

    Page History Explorer (PHE) is a system for interactive exploration of page histories by summarizing and visualizing evolution and past content of Web pages. PHE portrays the overview of page evolution by displaying past page snapshots on a two dimensional plane where one dimension is time and another one is a cummulative change degree of page content over time. The system also characterizes common, frequently appearing content of pages over time by term clouds. It lets users observe page histories from different viewpoints. In addition, PHE also enables comparison of histories of multiple pages. The figures below show system snapshots.

Past Web Browser

    We describe a new kind of a browser for viewing histories of Web pages and for navigating the past Web. Past Web is considered here as the accumulated set of past snapshots of pages that can be retrieved from Web archives or other available Web repositories (e.g. search engines caches).

  • The browser downloads past page snapshots of a given URL from available Web archives and presents them automatically along the timeline by pressing "PLAY" button. By pressing "STOP" button a user can stop the browsing. The speed or direction of viewing can be also adjusted.
  • Page evolution is visualized by detecting and animating content changes between consecutive past page snapshots. In this visualization deletion type content changes between consecutive page snapshots are first blinking and then dissappear from the page. The addition type changes appear on the page and blink for a short time to draw user attention.
  • There are navigation tools provided for spatio-temporal structure of the Past Web. User can use 2 forward, 2 back buttons or select page snapshots from the timeline or the list.
  • The browser enables filtered browsing by query input. In such a case only the changes containing the query will be animated. For example, a user can view the history of a news page from the viewpoint of a certain event, object etc.
